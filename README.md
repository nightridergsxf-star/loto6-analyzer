# Lottery Analyzer - ロト6 / ロト7 分析 & 予測ツール

ロト6 / ロト7 の過去全抽選データを多角的に統計分析し、スコアリングに基づいたおすすめ番号を生成する Python ツール & Web アプリです。Web 側ではヘッダーでロト6/ロト7 を切り替えて表示できます。

## セットアップ

### 必要環境
- Python 3.9+
- 必要パッケージ: `pandas`, `numpy`

```bash
pip3 install pandas numpy
```

### 実行方法

```bash
# JSON 生成（Web 版の前処理。両ゲームをまとめて更新）
python3 scripts/generate.py
python3 scripts/generate.py --game loto7   # 片方だけ

# CLI 版（ロト6 専用レガシー。互換維持のため残置）
python3 loto6_analyzer.py
```

`generate.py` は実行のたびに最新 CSV を自動ダウンロードして分析します。

## 分析内容（全13種）

※ 以下 `()` 内の数値は ロト6 / ロト7 の順。パラメータは `scripts/game_config.py` の GameConfig で切替されます。

### 基本分析
| # | 分析 | 内容 |
|---|------|------|
| 1 | 出現回数・出現率 | 各番号 (1〜43 / 1〜37) の出現回数と理論値との偏差 |
| 2 | ボーナス数字 | ボーナス数字 (1個 / 2個) の出現傾向 |

### パターン分析
| # | 分析 | 内容 |
|---|------|------|
| 3 | 連番ペア | 連番(例: 5,6)がどれくらい出るか |
| 4 | 奇数・偶数バランス | 毎回の奇数/偶数の比率分布 |
| 5 | 高低バランス | 低い数字 (1〜21 / 1〜18) vs 高い数字の分布 |
| 6 | 十の位グループ | 1桁台/10番台/20番台/30番台/40番台 の偏り（ロト7 は40番台なし） |
| 7 | 末尾数字 | 末尾0〜9の出現分布 |

### 時系列分析
| # | 分析 | 内容 |
|---|------|------|
| 8 | ホット/コールドナンバー | 直近30回でよく出る番号・出ない番号 |
| 9 | 出現間隔(インターバル) | 各番号が何回おきに出るか、出遅れ番号の検出 |
| 10 | 前回引っ張り | 前回の当選番号が次回にも出る確率 |

### 高度な分析
| # | 分析 | 内容 |
|---|------|------|
| 11 | 合計値分析 | 当選 (6つ / 7つ) の合計値の分布と最適範囲 |
| 12 | 月別傾向 | 月ごとの出やすい番号TOP5 |
| 13 | ペア相関 | よく一緒に出る番号ペアの検出 |

## 予測ロジック（v2: テーマ別スコアリング）

### 6つのスコア要素

| 要素 | 説明 |
|------|------|
| 全期間出現率 | 理論値との比較 |
| 直近ホット度 | 直近30回の出現頻度 |
| 出遅れスコア | 平均間隔との比較（ソフト化: 段階的に加点） |
| 前回引っ張り | 前回の当選番号にボーナス |
| ペア相関 | 強いペアを持つ番号に加点 |
| 時間減衰 | 新しい出現ほど重く評価（指数減衰） |

### 6つのテーマ

テーマごとにスコアの重みが変わり、"違う人格"の予測を生成:

| テーマ | 特徴 |
|--------|------|
| 🔥 ホット追従型 | 直近の流れに乗る。勢いのある番号を重視 |
| ❄️ コールド反発型 | 出遅れ番号の揺り戻しを狙う |
| ⚖️ バランス重視型 | 全要素を均等に評価。安定志向の王道 |
| 🎯 中央値集中型 | 合計値を最頻帯に寄せる狙い撃ち |
| 🃏 ワイルドカード型 | 揺らぎ最大。攻めの構成 |
| 🪞 逆張り型 | 設計側の視点で読む。誕生日圏外/キリ番回避/連番回避で人のバイアスを外す |

### 選出方法

- **確率抽選（ルーレット選択）**: スコアをソフトマックスで確率変換し、毎回異なる結果を生成
- **ソフトスコア評価**: 合計値・奇偶・高低・十の位の分散を「点数化」（ハードフィルタではなく加減点）
- **温度パラメータ**: テーマによって揺らぎの大きさを調整

## データソース

- ロト6: [KYO's LOTO6](https://loto6.thekyo.jp/) の CSV（Shift-JIS → UTF-8変換）/ 全2094回分
- ロト7: [KYO's LOTO7](https://loto7.thekyo.jp/) の CSV（Shift-JIS → UTF-8変換）/ 全672回分
- データ生成: `python3 scripts/generate.py [--game loto6|loto7|all]`

## 注意事項

このツールは統計分析に基づく参考情報です。ロト6 / ロト7 は完全なランダム抽選であり、過去のデータから未来を予測することは理論上不可能です。あくまで楽しみとしてご利用ください。
